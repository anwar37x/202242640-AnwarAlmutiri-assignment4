# Personal Portfolio – Assignment 2

## Overview
This project is an improved version of my personal portfolio website from Assignment 1.  
It includes interactive features, better user experience, and dynamic content using JavaScript.

## Features
- Responsive design (Desktop, Tablet, Mobile)
- Dark mode with localStorage (remembers user preference)
- Project filtering (Dynamic Content)
- Form validation with user feedback messages
- Smooth animations and hover effects
- Skills section (Technical & Soft Skills)

## Technologies Used
- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript
- Font Awesome (icons)

## Interactive Features
- Users can filter projects using buttons (UI / VB / All)
- Dark mode is saved using localStorage
- Contact form validates input and shows feedback messages

## How to Run
1. Download or clone the repository
2. Open `index.html` in your browser
3. Use the navigation bar to explore sections
4. Try filtering projects and toggling dark mode
