# Personal Portfolio – Assignment 3

## Project Overview
This project is an advanced version of my personal portfolio website developed for Assignment 3.  
It builds on Assignments 1 and 2 by adding advanced JavaScript functionality, external API integration, state management, and improved user experience.

The website presents my profile as a Software Engineering student, highlights my projects and skills, and dynamically displays my live GitHub repositories.

---

## Main Features

### Portfolio Sections
- About Me
- Projects
- GitHub Repositories
- Skills
- Contact

### Interactive Features
- Filter projects by category
- Sort GitHub repositories (A-Z / Z-A)
- Dark mode toggle with saved preference
- Contact form validation with feedback messages
- Smooth scrolling and fade-in animations

### API Integration
The website uses the GitHub API to fetch and display my public repositories dynamically.

### State Management
Dark mode preference is stored using localStorage and restored when the page reloads.

---

## How to Run Locally

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.

No installation is required.

---

## How to Use the Website

1. Use the navigation bar to move between sections.
2. In Projects, filter projects using category buttons.
3. In GitHub section, sort repositories using sort buttons.
4. Toggle dark mode using the moon button.
5. Use the contact form and submit valid data.
