# AI Usage Report

## Overview

AI tools were used during the development of Assignment 4 to support learning, improve productivity, and enhance the overall quality of the project.

The main tool used in this assignment was ChatGPT.

---

## Tools Used

- ChatGPT

---

## How AI Was Used

AI assistance was used in several parts of the project, including:

- Generating ideas for advanced portfolio features
- Improving website layout and user interface design
- Assisting with JavaScript logic and debugging
- Implementing project filtering functionality
- Improving contact form validation
- Assisting with GitHub API integration
- Suggesting responsive design improvements
- Reviewing and refining code structure
- Helping write project documentation

---

## Benefits of Using AI

Using AI provided several advantages:

- Faster problem solving
- Better understanding of JavaScript concepts
- Improved code readability and structure
- Easier debugging process
- Better UI/UX decisions
- More professional documentation writing
- Faster implementation of advanced features

---

## Challenges Faced

Some challenges were encountered while using AI tools:

- Some generated code required modification
- Certain suggestions needed simplification
- Not all solutions matched the project directly
- Manual testing was still required after implementation

---

## Risks of AI Usage

Potential risks of AI assistance include:

- Incorrect or incomplete code suggestions
- Over-reliance without understanding concepts
- Security/privacy risks if code is copied blindly
- Using code that does not match assignment requirements

To avoid these risks, all AI-generated content was reviewed, edited, tested, and improved manually before submission.

---

## Example Prompts Used

Examples of prompts used during development:

- Help me improve my portfolio website design
- Add JavaScript project filtering to my website
- Explain how to use localStorage for dark mode
- Help me fetch GitHub repositories using API
- Improve my contact form validation
- Review my README file professionally

---

## Learning Outcomes

Through responsible AI use, I learned:

- Better DOM manipulation using JavaScript
- How external APIs work in front-end projects
- Using localStorage for user preferences
- Responsive design improvements
- Cleaner code organization
- Better debugging methods
- Improved documentation writing

---

## Responsible Use and Modifications

All AI-generated suggestions were treated as drafts, not final answers.

I reviewed every suggestion, modified code when necessary, tested functionality manually, and ensured I understood how each feature worked before using it.

AI was used as a learning assistant and development support tool, not as a replacement for my own work.

---

## Conclusion

AI tools helped accelerate development and improve project quality while also supporting my learning process throughout the assignment.
